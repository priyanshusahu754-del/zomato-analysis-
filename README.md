## 📌 Project Overview

This project performs an in-depth **Exploratory Data Analysis (EDA)** on the **Zomato Restaurant Dataset** to uncover valuable insights about restaurants, customer ratings, cuisines, pricing, online delivery, table booking, and city-wise trends.

The primary objective of this project is to clean, explore, visualize, and understand the dataset before applying Machine Learning techniques.

---

## 🎯 Objectives

- Understand the structure of the dataset
- Perform data cleaning and preprocessing
- Handle missing values
- Explore numerical and categorical features
- Identify trends and hidden patterns
- Visualize important relationships
- Generate business insights for restaurant analysis

# 📈 Key Insights

Some important observations obtained from the analysis include:

- Most restaurants belong to a limited number of major cities.
- Online delivery is available only for a subset of restaurants.
- Higher-rated restaurants generally receive more customer votes.
- Certain cuisines dominate the restaurant market.
- Restaurants with table booking often have comparatively better ratings.
- Most restaurants fall within the mid-price category.
- Customer ratings are concentrated between average and good ratings.
